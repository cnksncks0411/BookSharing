<도서관리>
도서 대분류별 누적 대출수와 가장 많이 대출된 도서의 정보 추출 -> 분야별 베스트 셀러 구분
성별 별로 가장 많이 대출된 도서의 정보 추출 -> 성별 별 베스트셀러
현재 보유 책 수량이 얼마나 되는지 대분류별로 출력 -> 카테고리별로 도서 밸런스 조절
대출이 가장 많이 발생한 소분류의 소분류명 3순위까지 추출 -> 도서 분야 트렌드 확인

도서목록 table 내 모든 도서의 대출비용을 10% 차감하는 커서 정의
도서공유등록 table에 새로운 도서를 입력하는 프로시저 생성 등록도서번호는 시퀀스 이용(1001부터 1씩 증가)
도서목록 table에서 도서명에 입력한 단어를 포함하는 도서정보를 출력하는 프로시저 생성
회원이 공유한 도서가 도서목록에 입력될 시 해당 도서의 원가와 훼손도를 기준으로 계산하여 회원의 보유포인트를 증가하는 트리거 생성
입력받은 기간(숫자)으로부터 오늘 날짜까지 등록된 도서 출력하는 프로시저
대출비용산정기준표로 도서 목록의 도서 등급을 처리하는 커서 정의
도서공유등록 테이블에서 도서공유등록일자를 기준으로 요일별 도서등록 회원수 추출하는 프로시저 생성

<회원관리>
회원의 등급별 각 인원수와 평균 공유도서수, 평균 누적결제금액, 평균 대출횟수를 추출 -> 등급별 통계, 등급 적합도 판단
8월 한 달 동안 책을 가장 많이 빌린 20명의 이름, 연락처, 회원등급을 출력해라 -> 대출 랭킹
회원 점수가 0 이하인 회원의 대출가능여부를 false로 변환 -> 연체가 심한 사람에 대한 패널티 부여
회원 등급이 다이아인 사람의 기본정보를 조회하시오 -> 혜택 지급 목적

입력한 회원 테이블의  회원점수를 변경할 때, 그 기준 (회원등급)테이블의 범위를 벗어날 경우 메세지와 회원정보를 출력하는 프로시저 생성
회원정보의 회원점수를 자동으로 계산해주는 트리거 생성
입력한 나이에 해당하는 회원에게 500포인트 지급하는 프로시저 생성

<연체>
연체일이 14일 지난 도서번호, 도서명, 회원이름, 연락처를 추출 -> 연체가 심각한 경우 회원에게 연락

도서대출 table에서 반납만기일을 넘긴 도서를 연체관리 table에 입력하는 커서 정의
반납 만기일 초과일 만큼 연체일, 연체금 변경하는 트리거 생성

<배송>
배송 희망일이 오늘인 도서번호, 도서명, 이름, 연락처, 주소 추출 -> 당일에 배송해야 할 도서 확인


<대출>
최근 3개월 간 대출이 1건도 일어나지 않은 도서의 대출비용을 15% 삭감 -> 워스트 셀러 대출 할인
최근 6개월 간 대출이 1건도 일어나지 않은 도서를 도서목록에서 삭제 -> 워스트 셀러 삭제

회원이 도서대출 신청 시 해당 도서의 대출가능 여부를 N으로 변경하고, 대출비용만큼 회원의 보유포인트를 차감하는 트리거 생성

<반납>
도서 반납만기일이 내일인 도서의 도서번호, 도서명, 회원이름, 연락처 추출 -> 반납 알림
금일 반납된 도서의 도서정보를 추출 > 재입고알림
반납율을 계산하라

반납수거 시(반납수거table에 데이터 입력시) 해당 도서를 도서반납 table 목록에서 삭제하고, 도서목록 table에서 해당 도서의 대출가능 여부를 Y로 변경하는 트리거 생성

<결제>
8월 한 달 동안 고객이 각 결제 수단을 얼마만큼 사용했는지, 총 금액은 얼마인지 사용 빈도수가 높은 순서대로 출력 -> 수익관리 및 인기 수단 확인
연령대 별 인원수와 가장 많이 사용된 충전 수단 추출 -> 통계 확인
도서목록 중 대출비용이 가장 저렴한 도서 보다 적은 포인트를 보유한 회원정보 추출 -> 결제 알림
월 별로 회원들의 공유로 지급된 포인트 총 합계와 결제를 통해 지급된 포인트의 총 합계 비율을 추출 -> 포인트 지급 비율 계산

포인트충전시 수단이 카드결제라면, 카드결제내역 table에 정보를 입력하는 프로시저 생성
결제 발생하면 포인트 변경하는 트리거 생성

<직원>
최근 세 달 간 수거및 배송을 가장 많이 한 사원의 급여를 3% 증가 -> 직원 급여 인상
수령일이 주말이라면 해당 배송 담당자에게 건당 3천원 보너스 지급 -> 우수직원 보너스 지급

수령 및 수거일이 주말(토, 일)이라면 해당하는 배송 담당자의 커미션을 3000원씩 누적하여 더하는 커서 정의
오늘이 1일이라면, 모든 사원의 보너스를 0으로 변경하는 커서 정의
부서의 커미션 변경
부서번호를 입력 받아 해당 부서의 사원 리스트를 출력하는 프로시저
새로운 사원정보 입력하는 프로시저
입력된 부서 사원들 급여 인상
급여값 입력받아 세금(3.3%)을 제한 실수령액 계산하는 함수
사장을 제외한 모든 사원의 급여를 10% 삭감하는 커서 정의
최저시급을 받는 사원이 존재할때 매년 사원의 급여가 당시 년도의 최저시급에 맞게 변경되는 트리거 생성

<이벤트>
2021년 8월 10일부터 8월 17일기간 동안 결제 금액 50000원 발생시, 해당 회원에게 포인트 10% 추가 지급 -> 결제 이벤트
오늘이 문화의 날(10월16일)이라면, 모든 도서 대출 비용을 10% 할인해 준다 -> 문화의날 이벤트
생일인 회원에게 등급별로 (2000,1500,1000,1000) 포인트 지급 -> 생일 이벤트
